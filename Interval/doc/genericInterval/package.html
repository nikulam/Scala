    Each instance of the class GenericInterval represents an interval -- an inclusive range -- on a scale.

    Each instance of the class GenericInterval represents an interval -- an inclusive range -- on a scale. An interval has a "start" and an "end", represented as objects of type T. An interval always contains at least a single item.

    An interval object may be used represent a range of just about anything that has an Ordering.

    An Interval object is immutable after it has been created. That is, its state can not be changed in any way.
